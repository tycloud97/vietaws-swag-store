# Viet-AWS Swag Store

### Introduction

- This project was built for the purpose of demo an AWS solution.
- We will focus on providing a solution based on the actual use case of an e-commerce website.

## Quick start

```
yarn
yarn start
```

## Deploy

```
yarn build
cd iac
npm i
npm run deploy
```
